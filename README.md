# Тестовое задание для Mindbox

## Задача № 1

### Условие

Напишите на C# библиотеку для поставки внешним клиентам, которая умеет вычислять площадь круга по радиусу и треугольника по трем сторонам. 
Дополнительно к работоспособности оценим:
- Юнит-тесты;
- Легкость добавления других фигур;
- Вычисление площади фигуры без знания типа фигуры в compile-time;
- Проверку на то, является ли треугольник прямоугольным.

### Структура проекта

- Mindbox.sln - solution-проекта:
   - Mindbox.Package - nuget-пакет для внешних клиентов;
   - Mindbox.Client - эмуляция внешнего клиента;
   - Tests - папка с тестами для solution:
      - Mindbox.Package.Tests - тесты nuget-пакета Mindbox.Package;
   - task2.sql - sql-скрипт для решения задачи № 2.

## Задача № 2

### Условие

В базе данных MS SQL Server есть продукты и категории.
Одному продукту может соответствовать много категорий, в одной категории может быть много продуктов.
Напишите SQL запрос для выбора всех пар «Имя продукта – Имя категории».
Если у продукта нет категорий, то его имя все равно должно выводиться.

### Ответ

```sql
SELECT p."Name" as "ProductName", c."Name" as "CategoryName"
FROM "Products" p
LEFT JOIN "ProductCategories" pc ON p."Id" = pc."ProductId"
LEFT JOIN "Categories" c ON c."Id" = pc."CategoryId";
```

### Пример

#### Исходные данные

Таблица `Products`:

| Id | Name      | Price |
|----|-----------|-------|
| 1  | Телевизор | 75000 |
| 2  | Ноутбук   | 90000 |
| 3  | Стол      | 5000  |
| 4  | Молоко    | 100   |
| 5  | Футболка  | 1100  |

Таблица `Categories`:

| Id | Name                   |
|----|------------------------|
| 1  | Электроприборы         |
| 2  | Компьютеры             |
| 3  | Мебель                 |
| 4  | Еда                    |
| 5  | Короткий срок хранения |

Таблица `ProductCategories`:

| ProductId | CategoryId |
|-----------|------------|
| 1         | 1          |
| 2         | 1          |
| 2         | 2          |
| 3         | 3          |
| 4         | 5          |
| 4         | 4          |
